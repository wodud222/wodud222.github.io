HTML기초와 태그


● HTML이란?

HTML은Hyper Text Markup Language란 말의 약자로, 하이퍼텍스트 문서를 만들기 위한 언어.

홈페이지를 만드는 프로그래밍 언어임


HTML은 인터넷상에서 문서 교환을 위해 만든 마크업(Markup) 표준 언어로서, 문서의 형식을 정의하게 된다.

즉 문서의 제목, 타이틀, 단락, 목록뿐만 아니라, 문서의 글자 모양과 폰트까지를 정의할 수 있다.

보통 HTML 문서는".html"의 확장자를 갖는다.




● HTML5란?

html5란 HTML의 최신 버전의 이름임

html5는 새로운 웹표준이며, HTML5하면 HTML5의 태그와 CSS3, 자바스크립트 모두를 의미하며

기존 웹은 Active X등의 플러그인 없이 스스로 동영상이나 음악을 재생할 수 없었으나, HTML5는 스스로 재생이 가능
웹브라우저 플러그인 없이 멀티미디어 컨텐츠를 실행해 보고자하는 것이 목적

시멘틱 웹을 구현, 지능형 웹을 의미


[ 액티브 X란 : 웹브라우저를 통해 동작하는 배포용 응용프로그램을 만들 수 있게 도와 주거나, 웹 사이트의 기능을 강화하거나 새로운 콘텐츠를 추가할 때 쓰는 웹 브라우저용 플러그인 (ex) 공인인증서 ]


[ 시멘틱 웹이란 : 컴퓨터가 사람을 대신하여 정보를 읽고 이해하고 가공하여 새로운 정보를 만들어 낼 수 있도록 이해하기 쉬운 의미를 가진 지능형 웹. (ex) 장애인을 위한 읽기 기능 등 ]

● 태그(tag)란?

태그(TAG)는 HTML 를 이루는 개개의 명령어, 이러한 TAG명령어들이 모여 HTML문서를 완성

태그란 약속된 언어(명령어)


예)     신문이.. 그 안에 글자와 사진, 한글, 영어 등이 조합 되서 하나의 신문이라는 게 만들어 지듯이.

        HTML문서는 글자, 사진이 조합된 신문의 한 페이지, 또 신문 한 페이지들이 모여 전체 신문이 완성되는 것처럼

        HTML문서는 여러 개의 웹페이지가 만들고 모아져 하나의 홈페이지 완성된다고 생각하면 됨



① <head></head>사이에 들어갈 수 있는 태그들

<meta />

meta 태그란 자신의 홈페이지에 관한 정보를 담아놓는 장치이며, 웹페이지에 관한 정보를 브라우저에 제공하는 역할을 함


■ 웹문서 언어 설정
<meta http-equiv="Content-Type" content="text/html;charset=UTF-8" />
웹 작업시 영어 이외의 다른 언어, 다국어 형태의 작업 시,
그리고 대부분의 css 및 스크립트 프레임워크들이 UTF-8로 인코딩되고 있기 때문에 euc-kr보다는 UTF-8사용권장

■ 일정 시간 경과 후 자동으로 다음 페이지 로딩하기
<meta http-equiv="Refresh" content="5;URL=http://www.naver.com" />


■ edge라는 의미는 최신버전에 맞추어서 화면이 나오게 하라 라는 뜻
<meta http-equiv="X-UA-Compatible" content="IE=edge">
